#  Employee Attrition Prediction

This project predicts whether an employee will leave the company or stay using Machine Learning models.

The dataset includes employee details such as age, experience, salary level, city, and job conditions.


##  Problem Statement
To predict employee attrition (LeaveOrNot):
- 0 → Stayed
- 1 → Left



##  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn



##  Dataset
- Total Records: 4653  
- Features: 9  
- No missing values after cleaning  
- Duplicate values removed  



## Data Preprocessing
- Removed duplicate records  
- Encoded categorical variables using One-Hot Encoding  
- Scaled numerical features using StandardScaler  
- Train-Test Split (80%-20%)  



##  Models Used
- Decision Tree Classifier  
- Random Forest Classifier  



##  Results
- Decision Tree Accuracy: 70.71%  
- Random Forest Accuracy: 73.78%  

 Random Forest performed better due to better generalization.


## Conclusion
This project helps understand employee behavior and supports HR decisions to reduce attrition.
